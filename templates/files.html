Snake game in python

This script sets up a simple Snake Game where a snake moves around the screen, eating food to grow longer while avoiding collisions with the screen's borders and itself. The game uses the turtle module for the graphical interface and random for placing food at random locations.

4 in a row game in python

This Python script implements a basic Connect Four game using the Pygame library for graphical rendering and NumPy for managing the game board.The script simulates a two-player Connect Four game, where players take turns dropping colored discs into a 7x6 grid. The first player to align four discs vertically, horizontally, or diagonally wins the game.

Movie reccomendation app

This Python script is a basic implementation of a content-based movie recommendation system using the TF-IDF (Term Frequency-Inverse Document Frequency) method to recommend movies based on their plot overviews. The system calculates the similarity between movies using cosine similarity and recommends movies that are most similar to a given title.
